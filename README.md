# tokimo-package-pt-search

PT site torrent search and user info scraping library.

## Features

- Multi-site PT search with configurable site definitions
- NexusPHP / Gazelle / Unit3D parser support
- User info scraping (ratio, uploaded, downloaded)
- RSS feed parsing

## Add to Cargo.toml

```toml
tokimo-package-pt-search = { git = "https://github.com/tokimo-lab/tokimo-package-pt-search" }
```
